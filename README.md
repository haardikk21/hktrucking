# HKTrucking

## Features
Whirlpool Hashing
ZCMD Processing
Y_INI for saving
5 Different Teams
Unlimited Missions

## Teams
Trucker
Busdriver
Taxidriver
Pilot
Boatman

## Modifying the code
You need to customize the registration questions and the mission locations to work with your gamemode.
Read LOCATIONS.md to learn more
